# Booking Appointment App (CRUDCrud Integration)

This is a simple Booking Appointment web app where user details are stored in the cloud using CRUDCrud API instead of local storage.

## Features
- Add user details (Name, Email, Phone)
- Data is stored using Axios POST request
- Data is displayed on the screen after submission

## Technologies Used
- HTML
- JavaScript
- Axios
- CRUDCrud API

## How it works
When the form is submitted:
1. User details are collected
2. A POST request is sent to CRUDCrud API
3. The data is stored in the cloud
4. The same data is shown on the screen

## API Used
CRUDCrud: https://crudcrud.com

## How to run
1. Clone the repository
2. Open the project folder
3. Open `index.html` in browser
4. Enter details and submit

## Note
- CRUDCrud endpoint expires after some time
- You may need to generate a new API endpoint
